HAD100 DATASET
A Large-scale Dataset for Hyperspectral Anomaly Detection

Zhaoxu Li, Yingqian Wang, Chao Xiao, Qiang Ling, Zaiping Lin, Wei An

Description

Although many novel hyperspectral anomaly detection methods have been proposed in recent years, there is no large-scale dataset that can comprehensively evaluate the performance of these methods. HAD100 is a unified evaluation benchmark dataset for the hyperspectral anomaly detection task. This dataset contains 100 test scenes with various targets and backgrounds. The test scenes are downloaded from AVIRIS-NG and uniformly cropped to patches of size 64 ✕ 64. Furthermore, the HAD100 dataset provides two training sets which are downloaded from AVIRIS-NG and AVIRIS-Classic, respectively. The AVIRIS-NG training set contains 260 anomaly-free hyperspectral imamgs, while the AVIRIS-Classic training set contains 262 anomaly-free hyperspectral imamgs.

For more detail, please refer to our paper.
